What you get

Everything you wished
Claude Code showed.

⎇
Branch Awareness

Always know which branch you're on. brocode reads from git before launching so the right context is visible from the first keystroke.

◆
Live Model Display

See the exact model Claude Code is using in real time — Sonnet, Opus, Haiku — directly in the status bar. No more guessing.

⊞
Context Progress Bar

A live progress bar tracks your context window usage. Know when you're approaching limits before Claude Code silently truncates your history.

◈
Cost Tracking

Session and daily cost estimates calculated from Claude's token pricing, displayed live. Understand your spend without leaving the terminal.

Setup in seconds

Three steps.
That's it.

1
Install globally

One command. brocode and brocode-status are both added to your PATH.


npm install -g brocode
2
Run brocode

Use brocode anywhere you'd use claude. All flags and arguments pass through unchanged.


brocode --resume
3
See your metrics

The welcome box appears on launch. The status bar at the bottom of Claude Code updates live throughout your session.

Zero dependencies.
Pure Node.js.

brocode uses only Node.js built-ins — child_process, fs, and path. Nothing to audit, nothing to break.
