View on GitHub settings.json

Install

1
Clone and run the bootstrap script
git clone https://github.com/adarshkumar/claude-setup && cd claude-setup && bash install.sh
2
Fill in your API keys in ~/.claude/settings.json
"ANTHROPIC_API_KEY": "sk-ant-..." ~/.claude/settings.json
3
Launch Claude Code — plugins and MCP servers activate automatically
claude

MCP Servers

All servers use npx — no global installs required. Node.js 18+ needed.
